# 機能バックログ（将来の追加機能案）

> 現在未実装の機能アイデア一覧。優先度・実現性は今後検討。

| # | 機能名 | 詳細 |
|---|---|---|
| 1 | Teams 通知連携 | Incoming Webhook 経由でブリーフィングを配信 |
| 2 | フォルダ読み込み制御の強化 | 除外フォルダ指定機能（特定サブフォルダを走査対象から除外）、親フォルダ配下のサブフォルダを自動検知しチェックボックスで走査対象を選択できる GUI |
| 3 | デッドライン・アラート | frontmatter の `deadline` を読み取り、期限が近い／超過したタスクを警告。「あと2日」「3日超過」など緊急度付きで通知 |
| 4 | ナレッジ・コネクター（横断的関連づけ） | 異なるフォルダ・ファイル間の意外なつながりを LLM が発見して提示。「プロジェクトAの○○は学習中の△△と関連しています」のようなセレンディピティを演出 |
| 5 | TODO / アクションアイテム集約 | 各 MD に散在するチェックボックス（`- [ ]`）や「TODO」を横断的に収集・一覧化。LLM が優先度付きで要約 |
| 6 | 知識ギャップ分析 | ノートが薄い領域や最終更新が古い箇所を検出し、「この分野は最終更新が3ヶ月前です。最新動向を確認しますか？」と提案 |
| 7 | 過去の自分ダイジェスト | 過去のブリーフィング（`_briefings/` 内）を参照し、「1週間前／1ヶ月前にあなたはこれを調べていました」と振り返りを提示 |
| 8 | 要約・TL;DR 生成 | 長い MD ファイルの3行要約を生成。最近追加された長文ノートのダイジェストとしてブリーフィングに含める |
| 9 | トレンド変化検知 | 過去のブリーフィングと比較して「先週と比べてこのトピックの動きが急加速しています」などトレンドの変化を検知・通知 |
| 10 | 関連トピック横断クイズ | 異なるノートの内容を組み合わせた横断問題。「AZ-305 で学んだ ExpressRoute と、Terraform ノートのネットワーク設定の関係は？」のように複数トピックの理解を深めるクイズを LLM プロンプトで生成。精度にばらつきがあるため要検証 |
